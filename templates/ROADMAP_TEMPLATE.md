## Prioritization Framework

All items are scored using **RICE** (replace with your preferred framework if different):

| Factor | Scale | How we measure |
|--------|-------|----------------|
| **Reach** | Users/quarter | How many users will this affect in the next quarter? |
| **Impact** | 0.25 / 0.5 / 1 / 2 / 3 | Minimal / Low / Medium / High / Massive impact per user |
| **Confidence** | 50% / 80% / 100% | How sure are we about reach and impact estimates? |
| **Effort** | Person-weeks | Total dev effort to ship |
